example 5: XML loading

Este ejemplo, requiere un servidor web real - no va a trabajar con los archivos locales.

Esta página demuestra la carga dinámica de XML. Haga clic en la "carga xml 'situado en la parte inferior de la página para cargar" example.xml' actualmente en el marco del nodo seleccionado. Tenga en cuenta que al cargar un documento XML en un nodo se borra el contenido actual de ese nodo.

Por favor, examine el archivo XML. Los atributos' texto ',' Icono ',' openicon ',' color 'y' abierto 'se utilizan para la MooTreeNode constructor para cada uno de los nodos - cualquier otra atributos adicionales (como "alerta" en uno de los nodos en este ejemplo) se añaden a la del nodo "datos" objeto, incluido el atributo especial "carga". La "carga" atributo (opcional) especifica un archivo XML para cargar cuando el nodo se amplía..
